# BATTLE ORIGIN - Standalone Prototype

これは、会話で確定した範囲だけを実装した新規サイト用プロトタイプです。

## 実装済み

- 完全に別サイトとして動く構成
- Node.js + Express の静的配信
- タイトル画面
- 初期環境選択
  - 武人：実装済み
  - 殺し屋 / 流浪 / 軍人：名前だけ確定、詳細未設計のため選択不可
- 武人スタート
  - 武術都市
  - お試し場ロビー
  - 受付
  - 個室
  - マネキン
  - 複数の練習武器
  - プレイヤーが自分で触って感じる形式
  - 評価、点数、おすすめ表示なし
- 流派案内板
  - 素手系
  - ナイフ系
  - 刀系
  - 長物系
  - 重量武器系
  - 特殊武器系
- 戦闘本体の導入
  - 一人称2D
  - 自分の手元 / 武器を表示
  - ドラッグ操作で武器を動かす
  - マネキンに触れた時だけ軽い反応
  - ダメージ、成長、流派効果、対人戦は未実装

## あえて入れていないもの

未決定・曖昧な部分は入れていません。

- 殺し屋 / 流浪 / 軍人の開始地点
- 具体的な流派名
- 師範
- 道場の場所
- 入門条件
- 成長システム
- ステータス
- オンライン対戦
- マッチング
- ダメージ計算
- 部位別判定
- 勝敗条件

## 起動方法

```bash
npm install
npm start
```

起動後、ブラウザで以下を開きます。

```text
http://localhost:3000
```
